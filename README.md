# SCAPS-1D-generated-Strontium-antimony-iodide-dataset
# 📊 Strontium Antimony Iodide (Sr₃SbI₃) Perovskite Solar Cell Dataset 

This repository contains a comprehensive simulation dataset for a Strontium Antimony Iodide (Sr₃SbI₃)-based Perovskite Solar Cell (PSC). The dataset is generated using the SCAPS-1D simulation tool and is intended for research in photovoltaic device optimization and machine learning-based performance prediction.

# 🔢 Dataset Overview
Total samples: 15,550
Total features: 12
Target variable: Power Conversion Efficiency (PCE)

#📌 Input Features
Feature Name	Description
Absorber layer thickness	Thickness of the perovskite absorber layer (µm)
HTL thickness	Thickness of the Hole Transport Layer (µm)
ETL thickness	Thickness of the Electron Transport Layer (µm)
Temperature	Operating temperature (K)
Absorber layer defect density	Defect density within the absorber layer (cm⁻³)
Absorber layer acceptor density	Acceptor doping concentration in absorber (cm⁻³)
ETL/Absorber interface defect density	Interface defect density between ETL and absorber (cm⁻²)
HTL/Absorber interface defect density	Interface defect density between HTL and absorber (cm⁻²)
